메서드를 작성하는법
접근제한자 반환형 함수명(){}

<접근제한자>
1.public
- 모든곳에서 접근가능

2.private
- 현재 클래스에서만 접근 가능

3.default
- 현재 패키지안에서만 접근 가능

4.protected
- 상속관계에서만 접근 가능

<반환형>
반환하려고 하는 데이터의 타입 -> 기본자료형 뿐만 아니라
String(클래스) 처럼 우리가 만든 클래스의 타입도 가능하다.
반환을 안하면 void

<함수명, 메서드명>
함수,메서드 이름을 지정함

<매개변수>
함수가 호출될때 값을 전달받아 저장하는 지역변수

함수를 호출 -> 함수가 선언된 곳으로 가서 작업을 수행
-> 수행을 마치고 나면 함수를 호출한 곳으로 흐름이 돌아온다.
반환된 값이 있다면 값을 가지고 돌아온다.
