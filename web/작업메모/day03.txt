CSS
- Cascading Style Sheet
- CSS는 HTML의 각 요소(element)의 style(design, layout etc)을 정의 하여
  화면(Screen) 등에 어떻게 렌더링하면 되는지 브라우저에게 설명하기 위한 언어이다.
- 색상, 폰트, 각 요소의 위치 변경, 백그라운드 및 간단한 애니메이션(transition) 등의
  효과를 주어 보다 보기 좋게 꾸며 주는 역할을 합니다.

CSS 선택자
1.태그의 이름으로 선택
h2{color:red;}

2.클래스 선택
- 특정 집단의 여러 요소를 한번에 선택할때 사용
.a1{color:red}

3.아이디 선택
- css를 적용할 특정 요소를 선택할 때 사용
#a1{color:red}

색상을 표현하는방법
css에서는 배경색, 글자색, 테두리의 색 등과 같이 생상을 다루는 경우가 자주있다.

inherit 
영단어로 상속이라는 의미이다.
스타일 값으로 inherit를 지정하면 부모 태그의 색상을 그대로 물려받는다.
css의 기본값이며 따로 color 속성을 지정하지 않는다면 자동으로 inherit로 지정된다.

컬러네임(color name)
red, green, blue

RGB 또는 RGBA 코드(Decimal)
- RGB코드는 색상을 표현하는 가장 단순한 코드이다.
(255, 255, 255)와 같이 3개의 숫자가 묶여있는 순서 쌍으로 정의된다.
각각 빛의 삼원색인 빨강,초록,파랑색을 의미한다.

-RGBA는 투명도를 포함한다
- 색상을 의미하는 RGB뒤에 투명도를 의미하는 알파값이 추가된 형태
16진수 컬로코드 (Hex RGB)
- 색상을 16진수로 표현하는 것으로 컴퓨터와 디스플레이의 발달로
  사용도가 매우 높아지고 있다.

글자를 표현하는 스타익
- HTML의 본질은 정보 전달이며, 정보 전달의 핵심은 텍스트이다.
- 따라서 CSS 에서는 텍스트를 아름답게 표현하기 위한 온간 도구들이 준비되어 있다.

3.vw,vh
브라우저 창의 해상도를 기준으로 삼는 길이의 단위
100vw는 브라우저의 화면 너비의 100%를 의미하며 30vw는 화면 너비의 30%를 의미한다.
vw,vh를 사용하면 화면의 크기에 상관없이 항상 일정한 비율로 화면에 컨텐츠를 표시 할 수 있다.

요소의 배치
display:gird 속성
grid 스타일은 css를 사용해 일종의 격자를 그리는 테크닉입니다. 그리드를 적용하려면 <div> 태그를 대상으로 "display: grid"
스타일을 적용하면 된다.

그리드 적용시 주의사항
그리드를 적용할 때 반드시 격자의 내용물이 될 구성요소를 한 덩어리의 태그로 표현해야한다.
애매할 경우 <div>로 감싸버리면 된다.
