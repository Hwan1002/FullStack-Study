<자바스크립트 코드를 실행하는법>
크롬 브라우저의 개발자 모드의 콘솔은 자바스크립트의 실행을 위한
인터프리터가 내장되어 있으므로, 콘솔에서 바로 자바스크립트 코드를 할수있다.

<데이터타입(자료형)>
typeof(데이터)
데이터의 자료형을 확인할 수 있다.

Number -> 숫자형태 정수,실수

String -> 문자형태 String

Boolean -> 참과 거짓의 표현

undefined -> 정수의 값이 할당되지 않았음을 알려준다.

String 문자를 표현하기 위한 자료형
